##My Blog

Coffee, LESS, and HTML create it.

Grunt watches it.

Jekyll assembles it.

Amazon S3 hosts it.

People read it.

###Install

    sudo npm install
    
###Watch

    grunt

###Build

    grunt build
    
###Deploy

    grunt deploy
